# ELLIE - Virtual Friend Chatbot 🤖

ELLIE is a simple conversational chatbot built with Python and ChatterBot for the Advanced Artificial Intelligence course.

## Features

- Friendly conversations
- Custom training data
- Easy to extend
- Terminal-based interface

## Technologies Used

- Python
- ChatterBot
- spaCy
- PyYAML

## Project Structure

```
.
├── chatbot.py
├── data/
│   └── conversation-data.yml
├── screenshots/
|   └── chatbotconversation.png
├── requirements.txt
├── README.md
└── .gitignore
```

## Installation

```bash
pip install -r requirements.txt
python -m spacy download en_core_web_sm
```

## Run

```bash
python chatbot.py
```
